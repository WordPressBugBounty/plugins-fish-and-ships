32 bedingte Auswahlmethoden für Versandregeln

Advanced Shipping Rates for WooCommerce verfügt über eine große Auswahl an Methoden:

Auf Produktdaten basierende Selektoren

Preis:

Preis des Produkts. Wenn es einen Verkaufspreis gibt, wird dieser verwendet.

Unabhängig von der WooCommerce-Einstellung "Preise mit Steuer" (kann mit oder ohne Steuer sein) werden die Preise der Produkte so übernommen, wie sie auf der Editionsseite eingegeben wurden.

Gewicht:

Wenn für die Produkte kein Gewicht festgelegt wurde, wird 0 genommen.

Volumen:

Der genommene Wert ist die Multiplikation von Breite x Höhe x Länge. Wenn die Größe nicht festgelegt wurde, wird der Wert 0 genommen.

Volumetric PRO,
Volumetrisches Set PRO:

Bei den auf dem volumetrischen Gewicht basierenden Methoden werden zwei Werte berücksichtigt: das tatsächliche Gewicht und das volumetrische Gewicht, und der höchste Wert wird verwendet. Wenn Sie das volumetrische Gewicht auswählen, erscheint ein neues Feld unterhalb der Regeltabelle, in das Sie den Faktor für das volumetrische Gewicht eingeben können. Sie sollten ihn von Ihrem Transportunternehmen erhalten.

HINWEIS: Das volumetrische Set berücksichtigt alle Volumenprodukte im Vergleich zu allen Gewichtsprodukten der Gruppe, im Gegensatz zum volumetrischen Set, das Gewicht und Volumenprodukte pro Produkt vergleicht und das Ergebnis anschließend summiert.

Mehr über volumetrische Gewichtsklassen erfahren Sie in diesem Beitrag.

Minimale Dimension,
Mittlere Dimension,
Maximale Dimension:

Sie berücksichtigen jeweils die kleine, mittlere und große Größe der Produkte. Es spielt keine Rolle, ob die Größen von klein nach groß oder von groß nach klein eingegeben wurden, bei einem Produkt von: 10x15x1 oder 15x1x10, die maximale Größe ist immer 15, die mittlere 10 und die minimale 1. Diese Methode nimmt die Produkte immer einzeln auf, unabhängig von der gewählten Gruppierungsmethode. Wenn für ein Produkt keine Größe festgelegt wurde, wird der Wert 0 genommen.

Länge+Breite+Höhe PRO:

Der Wert, der genommen wird, ist die Summe der Dimensionen. Bei einem Produkt von: 10x15x1, wird es sein: 10+15+1 = 26. Diese Methode nimmt die Produkte immer einzeln auf, unabhängig von der gewählten Gruppierungsmethode. Wenn für ein Produkt keine Größe festgelegt wurde, wird der Wert 0 genommen.

Länge+Größe PRO:

Der Wert, der genommen wird, ist das Ergebnis der Formel: Länge + 2 * Breite + 2 * Höhe der Produktabmessungen. Bei einem Produkt von: 10x15x1, wird es sein: 10+2x15+2x1 = 42. Diese Methode nimmt die Produkte immer einzeln auf, unabhängig von der gewählten Gruppierungsmethode. Wenn für ein Produkt keine Größe festgelegt wurde, wird der Wert 0 genommen.

Produktartbasierte Selektoren

In Kategorie,
NICHT in Kategorie PRO:

Sie können festlegen, auf welche Produkte die Regel zutrifft, basierend auf seiner Produktkategorie. Die Kategorie wird genommen, nicht die Kinder. Sie sollten auch die Kinder einbeziehen, wenn Sie eine Übereinstimmung wünschen.
Diese Methode nimmt die Produkte immer einzeln auf, unabhängig von der gewählten Gruppierungsmethode.

Tagged as,
NOT Tagged as PRO:

Sie können festlegen, welche Produkte auf der Grundlage der Produktkennzeichnung(en) der Regel entsprechen sollen.
Diese Methode nimmt die Produkte immer einzeln auf, unabhängig von der gewählten Gruppierungsmethode.

In der Versandklasse,
NOT In der Versandklasse:

Sie können festlegen, für welche Produkte die Regel gelten soll, basierend auf ihrer Versandklasse.
Diese Methode nimmt die Produkte immer einzeln, unabhängig von der gewählten Gruppierungsmethode.

Erweiterte Selektoren

Artikel im Warenkorb:

Die Anzahl der Produkte im Warenkorb wird übernommen, es sei denn, Sie haben die spezielle Aktion verwendet, die Produkte von den folgenden Regeln ausschließt, oder diese Option innerhalb der Versandboxen: In diesem Fall wird die Menge verringert.

Beachten Sie, dass das Kriterium Gruppieren nach hier verwirrend sein kann! Wenn Sie z.B. nicht gruppieren, werden die Produkte einzeln genommen, so dass die Menge zum Vergleich immer 1 ist. (siehe group-by).

Warenkorb insgesamt PRO
(inkl./exkl. MwSt.):

Der Zwischensummenpreis. Der Unterschied zwischen der Verwendung von Products Price (mit einer gruped-all-Strategie) und diesem Selektor besteht darin, dass wir hier den Zwischensummenpreis nach Anwendung der Coupons erhalten, d.h. den tatsächlichen Betrag des Warenkorbs vor Versandkosten.

Anzahl der Gruppen:

Die Anzahl der Gruppen, abhängig von der group-by Strategie. Ein Beispiel: Wenn sich im Warenkorb 10 Produkte aus zwei verschiedenen Versandklassen befinden, ist der Wert 2, wenn Sie group-by pro Versandklasse einstellen. Wenn Sie keine Gruppe festlegen, wird der Wert 10 sein, und wenn Sie group-by auf alle gruppiert einstellen, wird der Wert 1 sein.

Benutzerrolle PRO:

Sie können festlegen, welche Benutzerrollen auf diese Regel zutreffen (auch nicht angemeldete Benutzer).

Mehr über die Versandtarife nach Benutzerrollen erfahren Sie in diesem Beitrag.

Datum & Uhrzeit Selektoren

Wochentag PRO,
Monat PRO:

Sie können auswählen, an welchen Wochentagen/Monaten des Jahres die Regel gelten soll.

Tag des Monats PRO
(inkl./exkl. MwSt.):

Sie können einen Bereich von Tagen im Monat festlegen, auf den die Regel zutrifft.

Jahr PRO:

Sie können einen Bereich von Jahren festlegen, in dem die Regel greifen soll.

Datum PRO:

Sie können einen Datumsbereich festlegen, auf den die Regel zutrifft.

Tag des Jahres PRO:

Sie können einen Tagesbereich festlegen, auf den die Regel zutrifft. Anders als der Datumsselektor deckt dieser Selektor einen Jahreszeitraum ab.

Zeit PRO:

Sie können einen Zeitbereich festlegen, in dem die Regel greifen soll.

Selektoren für Lieferadressen

In Zonenregionen PRO,
NICHT In Zonenregionen PRO:

Sie können es auf die gleiche Weise wie die WooCommerce-Versandzonen verwenden. Nützlich, um die Erstellung von zu vielen Versandzonen zu vermeiden.

In ZIP/Postcodes PRO
NICHT in ZIP/Postcodes PRO:

Sie können es auf die gleiche Weise verwenden wie die WooCommerce-Versandzonen. Nützlich, um Regeln für die Nähe aufzustellen.

Sie können Wildcards und Bereiche auf die gleiche Weise wie WC verwenden. Mehr über Tarife nach Postleitzahlen erfahren Sie in diesem Beitrag.

Größer als / Größer als oder gleich & Kleiner als / Kleiner als oder gleich

Die meisten Selektormethoden verwenden Felder mit Mindest- und Höchstwerten.

Für alle diese Selektoren können Sie wählen, wie die MIN- und MAX-Werte verglichen werden sollen, indem Sie bei jeder Regel nach Bedarf zum entsprechenden Symbol wechseln.

In der folgenden Tabelle setzen wir den MIN-Vergleich auf GRÖSSER ALS 5€ und den MAX-Vergleich auf WENIGER ALS 30€.

4.99€ 5.00€ 30.00€ 30.01€

In der zweiten Tabelle haben wir den MIN-Vergleich auf GRÖSSER ALS 30€ und den MAX-Vergleich auf WENIGER ALS 100€ eingestellt, denn die gleichen Werte entsprechen nicht dieser Regel:

30.00€ 30.01€ 99.99€ 100.00€

Beachten Sie, dass 30,00€ mit Regel #1 übereinstimmen, weil der Vergleich im MAX-Feld WENIGER ALS ODER GLEICH ist, und nicht mit Regel #2 übereinstimmen, weil MIN den Vergleich mit GRÖSSER ALS durchführt.

Auf der anderen Seite entsprechen 100€ nicht der Regel #2, da der MAX-Feldvergleich WENIGER ALS lautet.

Die volumetrischen Berechnungen

Wie funktioniert das? Nehmen wir einen volumetrischen Gewichtsfaktor von 50 kg/meter3 an. Unser Shop-Beispiel ist so konfiguriert, dass es mit Zentimetern und Gramm arbeitet, umgerechnet ist das: 0,05 Gramm pro Kubikzentimeter, (hier haben Sie einen Online-Konverter Einheiten für Volumen/Gewicht):

Produkt Gewicht Größe Band Volumetrisches Gewicht Genommener Wert
Mantel 1500 g. 50x50x14 cm 35000 cm3 35000 * 0,05 =
1750 g.		 1750 g.
Laptop 2500 g. 40x46x18 cm 33120 cm3 33120 * 0,05 =
1656 g.		 2500 g.

Bei leichten Produkten wird als Wert das berechnete Volumengewicht genommen. Bei schweren Produkten wird der Gewichtswert genommen. Immer der höchste Wert.
