Autres domaines

En dehors des règles du tableau, il existe d'autres champs que nous expliquerons ci-dessous :

Titre de la méthode :

Il s'agit du nom des frais de port que votre client verra (si nous ne le modifions pas par une action spéciale).

Statut fiscal :

Faut-il appliquer des taxes sur les frais d'expédition?

Groupe global par : PRO

En décochant cette option, vous pouvez définir un critère de regroupement différent pour chaque sélection.

Groupe par [for all selectors]:

La façon dont les produits du panier seront regroupés pour analyser les conditions des règles (voir group-by)

Type de calcul :

Vous pouvez ici choisir d'appliquer le coût de toutes les règles correspondantes, uniquement la plus chère ou la moins chère.

Frais d'expédition minimum :

Dans le cas où une méthode d'expédition est applicable, vous pouvez définir le coût minimum pour cette méthode d'expédition.

Frais d'expédition maximum :

Le coût maximum que la méthode d'expédition ne peut en aucun cas dépasser.

Système de journalisation (écrire les journaux :)

Si vous l'activez, un journal de tous les calculs effectués pour cette méthode d'expédition sera stocké pendant 7 jours, ce qui vous aidera à comprendre ou à déboguer vos paramètres. Cette fonction est également utile pour nous demander de l'aide ou nous aider à résoudre une erreur que nous aurions pu commettre.

Nous pouvons l'activer pour tous les utilisateurs, ou seulement pour les utilisateurs ayant un rôle d'administrateur (recommandé si nous voulons déboguer un magasin en cours d'exécution).
