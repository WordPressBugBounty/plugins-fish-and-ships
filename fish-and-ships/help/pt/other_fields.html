Outros domínios

Para além das regras da tabela, existem outros campos que explicaremos a seguir:

Título do método:

Este é o nome do custo de envio que o teu cliente verá (se não o alterarmos com uma Ação Especial)

Situação fiscal:

Deves aplicar impostos sobre os custos de envio?

Grupo global por: PRO

Se desmarcares esta opção, podes definir um critério de agrupamento diferente em cada seleção

Agrupar por [for all selectors]:

A forma como os produtos do carrinho serão agrupados para analisar as condições das regras (ver Agrupar por)

Tipo de cálculo:

Aqui podes escolher entre aplicar o custo de todas as regras de correspondência, apenas a mais cara ou a mais barata.

Custo mínimo de envio:

Se o método de envio for aplicável, podes definir o custo mínimo para esse método de envio.

Custo máximo de envio:

O custo máximo que o método de envio não pode exceder em caso algum.

Sistema de registo (Escreve os registos:)

Se a activares, será guardado um registo de todos os cálculos para este método de envio no prazo de 7 dias, o que ajudará a compreender ou a depurar as tuas definições. Também é útil para nos pedires ajuda ou para nos ajudares a corrigir algum erro que possamos ter cometido.

Podemos activá-lo para todos os utilizadores ou apenas para os utilizadores com função de Administrador (recomendado se pretendermos depurar uma loja em execução).
